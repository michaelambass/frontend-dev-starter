# Frontend Development Starter

Development dependencies :

```
"babel-core": "^6.26.0",
"babel-loader": "^7.1.2",
"babel-preset-env": "^1.6.1",
"http-server": "^0.11.1",
"uglifyjs-webpack-plugin": "^1.1.8",
"webpack": "^3.10.0"
```

## Launch server
```
$ npm run server
```

## Development process
Watch files

```
$ npm run webpack-dev
```

## Production process
Use code compression

```
$ npm run webpack-prod
```
